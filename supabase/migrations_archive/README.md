## Baseline Generation Method

The baseline was generated by:

1. Introspecting live production database
2. Extracting all table definitions via `information_schema`
3. Extracting functions via `pg_get_functiondef`
4. Extracting policies via `pg_policies`
5. Extracting views via `pg_views`
6. Organizing into logical, readable sections

## Risk Mitigation

- **No data loss:** Baseline captures structure only
- **Rollback safe:** Archive preserves full history
- **Idempotent:** Uses `CREATE IF NOT EXISTS` / `CREATE OR REPLACE`
- **Version tracked:** Records all original migration versions
